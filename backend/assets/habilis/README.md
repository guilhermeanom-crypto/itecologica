# Assets Habilis Internalizados

Este diretorio concentra os assets metodologicos trazidos para dentro da `ITECOLOGICA`.

Escopo atual:

- `agents/`: prompts oficiais dos agentes Habilis
- `models/`: modelos de conhecimento e estruturas-base

Regra de uso:

- a origem oficial do sistema continua sendo a `ITECOLOGICA`
- os arquivos aqui sao copias internalizadas e versionadas
- os prototipos originais fora da ITECOLOGICA permanecem intactos
